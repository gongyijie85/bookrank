# BookRank 书榜 UI 优化变更日志

## v2.0.0 — 2026-06-26

### 整体优化（8大区域）

#### 1. 首页排行榜卡片
- 增加搜索导航栏（56px高度），替代独立搜索条
- 分类标签改为底部指示条样式（2px墨绿色）
- 排行榜改为2列网格，名次1-3使用24px serif墨绿色，4+使用14px灰色
- 卡片间距 8px → 12px，阴影柔化
- 元数据颜色变浅，描述文字缩小

#### 2. 书籍详情页
- 封面与标题间增加渐变过渡效果
- 简介行高 1.6 → 1.8，字号 14px → 15px
- 详情信息从2列网格改为单列列表
- ISBN增加复制按钮
- 新增"返回榜单"按钮（墨绿描边样式）
- 区块间距 24px → 32px

#### 3. 获奖图书详情页
- 布局与书籍详情页保持一致
- 奖项信息（名称+年份）使用徽章样式展示

#### 4. 奖项榜单页
- Tab样式改为底部指示条（与首页分类标签一致）
- 增加奖项徽章（墨绿底）和年份徽章（暖橙底）
- 分页按钮最小高度 44px
- 10本书拆分为2页展示

#### 5. 周报列表页（新增）
- 新建 `weekly-report-list.html` 独立页面
- 左侧周数/日历指示器，右侧箭头
- 中文日期格式（如"2025年1月第2周"）
- 空状态包含插图+引导文字

#### 6. 周报详情页
- 统计数据从2x2网格改为4列水平布局
- 排名变化增加箭头（↑绿↓红）
- 推荐书籍改为气泡样式
- 底部增加"分享周报"按钮

#### 7. 底部导航栏
- 激活状态：填充图标 + 墨绿色
- 未激活：线性图标 + 灰色
- 首页Tab图标稍大（28px vs 24px）
- 4个Tab页共享统一规范

#### 8. 通用优化
- 所有触摸目标最小 44x44px
- 底部安全区域间距适配iPhone
- 空状态统一为插图+引导文字+操作按钮
- 搜索图标集成到导航栏
- Lucide图标统一使用 `@1.8.0` 版本

### 技术变更
- 新增页面：`weekly-report-list.html`
- Lucide CDN 版本统一为 `@1.8.0`
- 底部导航图标名称统一为 `house`（非 `home`）
- 所有交互 `data-dom-id` 已验证匹配 `.design` 文件
