关于 BookRank

汇集全球畅销书榜与图书奖项，为读者提供权威的图书参考

项目介绍

BookRank 是一个专注于图书排行榜数据聚合与展示的网站。我们实时追踪 纽约时报畅销书排行榜，收录国际重要图书奖项获奖作品， 并提供出版社新书速递与每周榜单分析，帮助读者快速发现优质图书。

数据来源

纽约时报畅销书榜

实时同步 NYT Books API，覆盖精装小说、非虚构、漫画等 8 大分类

国际图书奖项

收录诺贝尔文学奖、布克奖、普利策奖等权威奖项历年获奖书单

出版社数据

聚合 Penguin Random House、HarperCollins 等六大出版社新书信息

技术栈

Python 3.13 Flask 3.1 SQLAlchemy PostgreSQL Jinja2 原生 JavaScript
