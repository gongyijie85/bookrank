## 说明

1. **数据来源**: 以上 ISBN-13 信息来自 Open Library API 查询结果

2. **版本分类**:
   - **精装版 (Hardcover)**: 硬皮封面，通常价格较高，更耐用
   - **平装版 (Paperback)**: 软皮封面，通常价格较低，更轻便
   - **大众平装版 (Mass Market Paperback)**: 小尺寸平装版，价格最低
   - **电子书 (eBook)**: 数字版本
   - **音频版 (Audio CD)**: 有声书版本

3. **缺失数据**: 部分图书在 Open Library 中没有完整的版本信息，标记为"-"

4. **使用建议**:
   - 使用 ISBN-13（13位数字），是当前国际标准
   - 不同版本的 ISBN 不同，购买时请注意区分
   - 同一本书可能有多个平装版 ISBN（不同出版社或不同时间出版）
