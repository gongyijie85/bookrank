### 3. 🚀 CI/CD 门禁 ✅

  - `flake8` 代码风格检查

  - 根目录测试 (非阻塞)

### 4. 🏗️ 统一异常体系 ✅
- 重构 `app/utils/exceptions.py`:
  - 新增: `ExternalAPIError`, `DataNotFoundError`, `ServiceUnavailableError`, `DatabaseError`, `TranslationError`
  - 新增: `safe_call(fallback, log_level)` 装饰器
  - 新增: `safe_service_call()` 工厂
- 更新 `app/utils/__init__.py` 导出

### 5. 🔧 main.py 关键函数改造 ✅
- `_get_books_for_category`: 用 `ExternalAPIError` 替代静默失败
- `index()`: 捕获 `ExternalAPIError` 并日志记录，不崩溃
- `awards()`: 从单个大 try/except 拆分为 3 个独立 try:
  1. 奖项列表查询 → 独立降级
  2. 年份列表查询 → 独立降级  
  3. 图书数据查询 → 独立降级（含 filter/sort）

### 6. 🖼️ 图片懒加载 ✅
- 全覆盖检查完成: 所有 list/grid 视图已有 `loading="lazy"`
- 详情页主图保留 `eager`（合理）

### 7. ⏰ APScheduler 替换 daemon 线程 ✅
- 从 `threading.Thread(daemon=True)` 迁移到 `APScheduler 3.11`
- 3 个任务配置:
  | 任务 | 类型 | 频率 |
  |------|------|------|
  | 周报启动检查 | DateTrigger | 延迟后一次性 |
  | 新书同步 | IntervalTrigger | 每14天 |
  | 封面同步 | DateTrigger | 延迟后一次性 |
- 特性: `coalesce=True`, `max_instances=1`, `misfire_grace_time=3600`
- `__init__.py` 添加 `atexit` 优雅关闭钩子
