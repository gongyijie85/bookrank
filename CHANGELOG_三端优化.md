# 三端前端优化变更日志

## 版本信息
- **版本号**: v1.2.0
- **修改时间**: 2026-07-06
- **优化范围**: 电脑端、平板端、手机端前端响应式优化

---

## 主要变更

### 1. 断点体系统一
**修改文件**: `static/css/base.css`, `static/css/components.css`

**变更内容**:
- 将原有 `1024px` 单一断点调整为三端断点体系：
  - **桌面端**: ≥1200px（固定侧边栏，多列布局）
  - **平板端**: 768px-1199px（可切换侧边栏，双列布局）
  - **手机端**: ≤767px（底部导航，单列布局）

**优化效果**:
- 平板端（iPad 等）获得专门适配，不再直接降级为移动端布局
- 侧边栏在平板端可通过按钮切换显示/隐藏
- 卡片网格在平板端显示 4 列，桌面端 6-8 列，手机端 2-3 列

---

### 2. 平板端布局优化
**修改文件**: `static/css/base.css`

**变更内容**:
- 侧边栏在 768px-1199px 区间默认隐藏，通过 toggle 按钮切换
- 主内容区占满宽度，充分利用平板屏幕空间
- 顶部导航在平板端显示完整文字标签（不隐藏）
- 侧边栏展开时添加半透明遮罩层，提升视觉层次

**优化效果**:
- 平板横屏/竖屏均有良好体验
- 侧边栏切换流畅，遮罩层点击可关闭

---

### 3. 手机端导航增强
**修改文件**: `static/css/base.css`

**变更内容**:
- 手机端（≤767px）顶部导航优化：
  - 导航链接仅显示图标，隐藏文字（节省空间）
  - 导航链接支持横向滚动
  - 语言切换和主题切换按钮尺寸优化（≥36px 触摸目标）
- 安全区适配：
  - 顶部导航适配刘海屏（`env(safe-area-inset-top)`）
  - 主内容区底部适配 iPhone 底部横条（`env(safe-area-inset-bottom)`）
  - 输入框字体强制 16px（防止 iOS 自动缩放）

**优化效果**:
- iPhone 全系列（含刘海屏、灵动岛）完美适配
- 触摸目标符合 44px 最小尺寸规范
- 导航操作更便捷

---

### 4. 组件响应式优化
**修改文件**: `static/css/components.css`

**变更内容**:
- **卡片网格**:
  - 桌面端（≥1200px）: 6-8 列
  - 平板端（768px-1199px）: 4 列
  - 手机端横屏（481px-767px）: 3 列
  - 小屏手机（≤480px）: 2 列
  - 超小屏手机（≤360px）: 1 列

- **筛选栏**:
  - 手机端（≤767px）: 垂直堆叠，全宽输入框
  - 平板/桌面端: 水平排列

- **模态框**:
  - 手机端（≤767px）: 全屏画廊模式，封面居中
  - 平板端（768px-1199px）: 单列布局，封面居中
  - 桌面端（≥1200px）: 双列布局（封面 + 详情）

- **按钮和表单**:
  - 手机端按钮最小高度 44px（触摸友好）
  - 输入框字体 16px（防止 iOS 缩放）

**优化效果**:
- 各尺寸屏幕下卡片排列合理，不拥挤不稀疏
- 模态框在手机端阅读体验更佳
- 所有交互元素符合触摸操作规范

---

### 5. 侧边栏切换优化
**修改文件**: `static/css/base.css`

**变更内容**:
- 侧边栏切换按钮（`.sidebar-toggle`）在 `<1200px` 时显示
- 桌面端（≥1200px）隐藏切换按钮，侧边栏始终可见
- 切换按钮位置固定在左上角，方便操作

**优化效果**:
- 平板端可自由控制侧边栏显示
- 桌面端不受影响，保持原有体验

## 技术细节

### CSS 断点对比
| 设备类型 | 旧断点 | 新断点 |
|---------|--------|--------|
| 桌面端 | ≥1025px | ≥1200px |
| 平板端 | 769px-1024px | 768px-1199px |
| 手机端 | ≤768px | ≤767px |
| 小屏手机 | ≤480px | ≤480px（保持不变） |

### 关键 CSS 变量
```css
/* 布局相关 */
--sidebar-width: 260px;
--top-nav-height: 64px;

/* 平板端侧边栏 */
@media (max-width: 1199px) {
    .sidebar {
        width: 80vw;
        max-width: 300px;
    }
}
```

### 安全区适配
```css
/* iPhone 刘海屏 */
padding-top: env(safe-area-inset-top, 0px);

/* iPhone 底部横条 */
padding-bottom: env(safe-area-inset-bottom, 0px);
```
